# Support

## Support Period

Mayflower Sandbox follows a rolling support model:

| Version | Status       | Security Updates Until |
|---------|-------------|----------------------|
| 0.2.x   | **Active**  | Until 0.4.0 release  |

We support the **current** and **previous** minor release with security patches.
When a new minor version is released, the version before the previous one
reaches end-of-life.

## Update Channels

### PyPI (Primary)

```bash
pip install --upgrade mayflower-sandbox
```

All releases, including security patches, are published to
[PyPI](https://pypi.org/project/mayflower-sandbox/).

### GitHub Releases

Release notes, SBOM artifacts, and distribution packages are available at:
https://github.com/mayflower/mayflower_sandbox/releases

### Security Advisories

Critical security updates are published via GitHub Security Advisories:
https://github.com/mayflower/mayflower_sandbox/security/advisories
